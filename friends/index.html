该模块的目的：通过直观展示最新友链文章，以减少访客在寻找所需文章过程中的时间浪费，提升效率，并且该模块能够提升友链文章的访问量，增加曝光，对于友站的SEO略有帮助。

✅友链交换须知 ꉂ(ˊᗜˋ)
  • Gitee Issue链接: https://gitee.com/justlovesmile/friendissue/issues
  • Github Issue链接: https://github.com/Justlovesmile/friend-issue/issues
为避免因添加友链，而重复部署网站，现无论是留言还是issue均默认通过issue方式添加到该页面。

友链模板如下：

申请要求：

  • 1.先友后链，申请友链请前先添加本博客，本站信息在上方模板
  • 2.头像尽量https，且贵站能够正常访问
  • 3.文章数量大于10篇，主要是为了确保您有动力和毅力维护博客
  • 4.同意本站通过爬虫订阅贵站最新文章标题和发布时间并展示在友链订阅页
  • 5.😀请一定要积极向上，且符合社会主义核心价值观

※ 为保证友链质量，申请的友链将经过筛选，通过审核将会收到我的回复邮件。原则上过去已添加友链不会轻易删除，如遇死链，将会暂时放入失联友链中；如遇严重违反上述申请条件的博客，将会被直接移除。
